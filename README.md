# Murmur

A privacy-first voice memo app built on Sia decentralised storage.

## Live Demo
https://murmur-five-alpha.vercel.app

## What it does
- Record, save and playback voice memos
- End-to-end encrypted — audio never touches a central server
- Backed by Sia decentralised storage via @siafoundation/sia-storage
- Recovery phrase gives full cross-device restore
- User accounts via Supabase auth

## Tech Stack
React + TypeScript + Vite + Supabase + Sia Storage SDK
